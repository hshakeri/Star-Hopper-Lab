A one-page guide for teachers · ages 9–16 · browser-based, offline-capable, no accounts

The design

Each world's physics is the hidden rule. Students run the loop Question → Collect → Chart → Predict → Test → Publish: they gather their own data (each jump auto-logs to a live notebook), read the chart, commit a prediction before the official test, and publish a one-sentence conclusion that a friendly peer reviewer gently questions. Experiments are driven by KidCode, a tiny real programming language (variables, loops, conditionals, averages, simple models).

Classroom notes

  • One mission ≈ 15–25 minutes; each ends with a publishable "journal article" — a ready-made exit ticket.
  • Charts model good practice: labeled axes, zero baselines, explicit break marks, honest rounding.
  • Badges reward science virtues (replication, skepticism, checking outliers) — never grinding.
  • Deterministic seeds mean two students with the same seed see the same "storm" — reproducibility as a superpower.
  • Works offline after first load; all progress is stored locally on the student's device.
